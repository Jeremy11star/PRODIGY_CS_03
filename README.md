# PRODIGY_CS_03
This is a Python project for the Prodigy InfoTech Cyber Security Internship. The program evaluates password strength. It gives feedback on missing requirements.

The program evaluates password strength. It gives feedback on missing requirements.

Criteria checked:

Minimum length of 8 characters.

Presence of uppercase letters.

Presence of lowercase letters.

Presence of digits (0-9).

Presence of special characters.

Technology used:

Python

Regular Expressions (re module)

The tool guides users to create strong, secure passwords.
